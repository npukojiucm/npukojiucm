## 🧩 Проекты

### 📦 Резервное копирование фотографий VK → Яндекс.Диск
Курсовая работа — консольная Python-утилита для резервного
копирования фотографий пользователя из VK в Яндекс.Диск.

- работа с внешними API (VK API, Yandex.Disk REST API)
- выбор изображений максимального размера
- формирование уникальных имён файлов
- сохранение метаданных в JSON
- progress bar для отслеживания выполнения

🔗 Репозиторий:  
https://github.com/npukojiucm/vk-to-yadisk-backup
